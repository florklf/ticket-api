## Description

API for a concert ticket selling app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Project documentation

1. Run the following command :
```bash
$ npx @compodoc/compodoc -p tsconfig.json -s
```
2. Go to http://localhost:8080/
